# keiba-research-mvp

競馬予想の回収率・期待値・資金曲線を研究するためのMVPツールです。

## 概要
独自のルールベース・スコアリングエンジンを用いてレースを評価し、期待値の高い「買い」候補を抽出します。
ダミーデータを使用して、システム全体の動作（レース表示、スコアリング、バックテスト）をローカルで確認できます。

## セットアップ手順

1. **Node.jsの確認**
   - Node.js v18以上がインストールされていることを確認してください。

2. **依存関係のインストール**
   ```bash
   npm install
   ```

3. **データベースの初期化とシード**
   ```bash
   npx tsx scripts/seed.ts
   ```

4. **開発サーバーの起動**
   ```bash
   npm run dev
   ```
   ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## 主要機能
- **ダッシュボード**: 総レース数や基本情報の表示。
- **レース一覧**: 直近3日間のレース情報をグリッド表示。
- **レース詳細**: 各馬のスコアリング結果、買い/見送り判定、判定理由の表示。
- **バックテスト**: 過去データ（ダミー）に基づいた的中率・回収率のシミュレーション。

## 技術スタック
- **Frontend/Backend**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Database**: SQLite (better-sqlite3)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## 仮定と制約
- **データ**: 現在は `scripts/seed.ts` で生成されるダミーデータを使用しています。
- **スコアリング**: 特徴量計算はランダムなダミー値に基づいています（実データ連携時に実装予定）。
- **券種**: 単勝のみを対象としています。
- **環境**: Pythonが利用できない環境を想定し、Node.jsフルスタックで構築されています。
